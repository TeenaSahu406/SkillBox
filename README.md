# SkillBox

SkillBox is a full-stack job portal that connects job seekers with recruiters. Candidates can create profiles, apply for jobs, and recruiters can post and manage job openings.

## Tech Stack

- React
- TypeScript
- Vite
- Express.js
- Supabase
- Gemini AI
- Nodemailer

## Installation

```bash
npm install
npm run dev
```
